什么是java？
java是一门高级编程语言。编程语言分为机器编程语言（指令）和高级编程语言。
高级编程语言是用人类能看懂的语言去编写程序，这样便于人表达自己的思想。
所以高级编程语言编写出来的程序，需要一个翻译来把人类语言翻译成机器语言。这个翻译就是编译器

java语言编写的程序，需要先通过编译器进行编译，才能被执行。

java能做什么？
1.用java语言能做桌面应用开发(JavaSE)
2.java语言能做企业级应用开发（JavaEE），后台服务器
3.java语言能做移动应用开发（Android），移动客户端
4.java语言能做大数据分析(hadoop)


Java开发环境：
JDK：java 语言开发工具包，包含java给开发者提供的API（类库）
JRE：java 程序的运行时环境，包含Java虚拟机。

java程序是可以跨平台运行的。所有的java程序都是运行在java虚拟机(JVM)之中的。


配置Java环境变量：
为什么需要配置环境变量?
配置环境变量的目的，是为了在命令行中的任意目录下都能够运行程序。因为在启动命令行的时候，命令行程序会先到系统的环境变量路径中读取所有已经配置的路径。在运行程序的时候，会现在当前目录查找该程序，如果当前目录没有该程序，则会到预读的环境变量中挨个查找。如果在预读的环境变量中也查找不到该程序，则会抛出错误提示。

java的源程序，都是以.java结尾的文本文件。



javac命令：启动java的编译器，帮助程序员把源程序编译生java虚拟机（JVM）能够执行的二进制文件。
执行格式：javac 源文件名
javac命令编译完源程序之后,会生成.class文件.这个class文件就是JVM能够执行的二进制文件.


java命令:启动java虚拟机,让java虚拟机执行一个java程序.
执行格式: java 类文件  注意类文件不要跟上.class


java中类的定义格式：
class 类名{

}

如果某个类想要被java虚拟机执行，那么这个类中则必须要有一个执行的入口方法：main方法。也就是说，main方法就是我们程序执行的入口
main方法的定义格式是固定的：
public static void main(String [] args){

}

综上所述，一个能被JVM执行的类应该是以下格式：
class 类名{
	public static void main(String [] args){

	}
}

大括号代表的是作用域，比如：类的大括号，说明的是类的作用域有多大，大括号内部的元素都属于类。
			    方法的大括号，说明的是方法的作用域，大括号内部的元素都属于方法。


方法中的元素，都叫语句，语句都以分号结束.

注意：程序中所有的标点符号，都应该是英文状态下的，不能用中文标点。




注释：用来给程序添加说明的，注释的文本在编译的时候是不会被编译器编译的
注释的方式:
//  单行注释，注释一行
/*
	
*/  多行注释，注释一段代码





System.out.print("");java的输出语句，双引号中是文本。想换行的话用System.out.println("")，或者在文本中加上\n
